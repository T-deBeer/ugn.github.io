Internet Protocols

  1. Internet Protocol(IP): Is explicitly designed as an addressing protocol, mostly used alongside TCP. TCP/IP is the most common protocol for connecting to a network.

  2. Transmission Control Protocol(TCP): Is a transport protocol used with IP to make sure transmission of packets are safe. Is used for communicating over a network, the message is split into packets and reassembled at the end destination.

  3. Hyper Text Transfer Protocol(HTTP): HTTP is used to transfer hypertext between two or more systems, hypertext is languages like HTML used to create a webpage. It loads web pages and displays the web interface for users, it helps applications communicate with users.

  4. Hypertext Transfer Protocol Secure(HTTPS): HTTPS is a lot like HTTP except the packets that are transferred are encrypted as well.

  5. File Transfer Protocol(FTP): It is used to transfer files between servers over the internet with the help of TCP and IP

  6. Post Office Protocol(POP): POP3 is a is used to receive incoming e-mail messages.

  7. User Datagram Protocol(UDP): UDP is a substitute for TCP and allows for loss-tolerating and low-latency connections.

  8. Simple mail transport Protocol(SMTP): SMTP is used to send outgoing e-mail messages.

  9. Secure Shell(SSH): Is a protocol that secures network services over unsecured networks.

  10. Secure Shell File Transfer Protocol(SFTP): Is when a connection is secured when a file is being transferred.
