WWW vs.The Internet

The internet and the World Wide Web are not the same thing. The World Wide Web uses the internet to pass over the data and information.You would basically use the World Wide Web to access the internet.

The internet is a global network while the World Wide Web is a lot of data and information. The World Wide Web is used to obtain the information from the internet. Computers connected over the network with internet can have different port names while the web have fixed ports like http (GeeksforGeeks, 2021).

The World Wide Web is all the data and pages you see on your computer and the internet is the collection of all the computers linked over the network. The World Wide Web is one of the applications built on the internet. The World Wide Web is used to surf the web using search engines and addresses but the internet can do a lot more than that.
